# 03 — Context preserved (no loss)

**Criterion:** (a) No still-valid existing memory was deleted or overwritten without a superseding source. (b) The high-signal facts present in the window actually made it into memory (recall).

**Why:** The opposite failure from fabrication — silent dropping. Context rot happens when real facts quietly vanish across runs.

**Pass:** Anything removed or replaced has a clear superseding source; spot-checked high-signal window items are present in memory.

**Fail:** A valid prior fact was overwritten with nothing in the window justifying the change; or an obviously important decision / stakeholder fact in the window is missing from memory.

**How to check:** Diff against the pre-run snapshot for deletions. Pick 5 high-signal items from the window and confirm each is captured somewhere in memory.
